Serie A Scout Index

Data-driven Player Ranking Model

A model designed to evaluate and rank Serie A players using independent performance metrics — corrected for opponent difficulty, stabilized with Bayesian Shrinkage, validated against real outcomes.

▸ 6 independent KPIs — Output, Centrality, Boost, Consistency (IQR-based), Age Index (AII), Physical Reliability (PRI), standardized with Winsorized z-scores
▸ 406 players · 21 matchdays · 5 contexts — Total, Home, Away, vs Top 6, vs Strong Defenses
▸ Statistically validated — Predictive backtest Spearman ρ=0.70, Bootstrap 95% CI, Kendall τ ranking stability, Cohen's d effect size
xG · xA · SOS-weighting Bayesian Shrinkage ✨ TPI Pro — AII + PRI Statistically Validated

Gli indici del sistema
Output Adj / 90'
(xG+xA)/min×90 pesato per SOS — corregge per la forza degli avversari
Centralità Offensiva
% produzione squadra che passa dal giocatore — con Bayesian Shrinkage
Team Boost Ratio
xG squadra CON / SENZA — quanto vale davvero per la squadra
Consistenza (IQR-based)
1 − IQR/mediana — robust to outlier matches, bounded [0,1]
⭐ AII — Age Impact Index
Curva gaussiana picco 27 anni — valore nel ciclo di carriera
💪 PRI — Physical Reliability
Disponibilità + infortuni + gravità — affidabilità fisica storica
✨ Novità — TPI Pro
TPI Pro — 6 Dimensioni di Analisi
Il TPI classico usa 4 dimensioni offensive. Il TPI Pro aggiunge Età Index (AII) e Affidabilità Fisica (PRI) per uno scouting a lungo termine più accurato.
TPI Pro = media( z_output, z_centralità, z_boost, z_consistenza, z_AII, z_PRI )
